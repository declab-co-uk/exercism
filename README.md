# Exercism Solutions: Elixir

This repo holds my synced solutions for the [Exercism Elixir track](https://exercism.org/tracks/elixir). 

## Background

I've been using Exercism as a daily habit to drill the fundamentals of Elixir. The focus here isn't just getting the tests to green, but really trying to internalize pattern matching, recursion, and Elixir's approach to concurrency before going deeper into Phoenix and LiveView.

Everything is synced directly via the Exercism CLI. Please find the problem descriptions and tests at the above link.
